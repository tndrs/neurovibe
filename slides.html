The Game Plan

Weekend 1:

Building the Monolith & Core Logic

  • Use TDD to create the core sentiment analysis logic.
  • Build a single, working Spring Boot application.
  • Ensure a solid foundation of tests.

Weekend 2:

Deconstruction & Automation

  • Break the app into two microservices.
  • Containerize everything with Docker.
  • Set up a Jenkins server for a full CI/CD pipeline.
